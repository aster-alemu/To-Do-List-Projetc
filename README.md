# Todo List with Add and Remove Function Project

It is a Todo list with CURD (create, update, delete) methods 

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

- Install Node.js
- Clone the repository 
- On the root directory, run 'npm install' on the terminal to install packages.
- Then well to go
